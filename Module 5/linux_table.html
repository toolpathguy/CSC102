Top 15 Linux Commands for Beginners

Command Description How to Use
ls Lists files and directories in the current directory. `ls` - Displays the contents of the current directory.
`ls -l` - Displays detailed information about files.
`ls -a` - Includes hidden files in the listing.
cd Changes the current directory. `cd folder_name` - Navigate to a specific folder.
`cd ..` - Move up one directory.
`cd ~` - Navigate to the home directory.
pwd Prints the current working directory. `pwd` - Displays the full path of the current directory.
mkdir Creates a new directory. `mkdir new_folder` - Creates a folder named "new_folder".
`mkdir -p parent/child` - Creates parent and child directories.
rm Removes files or directories. `rm file.txt` - Deletes a file named "file.txt".
`rm -r folder_name` - Deletes a folder and its contents.
`rm -i file.txt` - Prompts before deleting a file.
cp Copies files or directories. `cp file.txt backup.txt` - Copies "file.txt" to "backup.txt".
`cp -r folder1 folder2` - Copies a folder and its contents.
mv Moves or renames files or directories. `mv file.txt new_file.txt` - Renames "file.txt" to "new_file.txt".
`mv file.txt /path/to/folder` - Moves a file to a different folder.
cat Displays the contents of a file. `cat file.txt` - Displays the contents of "file.txt".
`cat file1.txt file2.txt` - Concatenates and displays multiple files.
echo Prints text to the terminal or writes to a file. `echo "Hello World"` - Prints "Hello World" to the terminal.
`echo "Hello" > file.txt` - Writes "Hello" to "file.txt".
touch Creates an empty file or updates the timestamp of an existing file. `touch new_file.txt` - Creates a file named "new_file.txt".
`touch -c file.txt` - Updates the timestamp without creating a new file.
chmod Changes file permissions. `chmod +x script.sh` - Makes "script.sh" executable.
`chmod 644 file.txt` - Sets read/write permissions for the owner and read-only for others.
chown Changes file ownership. `chown user file.txt` - Changes the owner of "file.txt" to "user".
`chown user:group file.txt` - Changes the owner and group of "file.txt".
grep Searches for a pattern in a file. `grep "error" log.txt` - Finds "error" in "log.txt".
`grep -i "error" log.txt` - Case-insensitive search for "error".
find Searches for files and directories. `find . -name "*.txt"` - Finds all ".txt" files in the current directory.
`find /path -type d` - Finds all directories in the specified path.
man Displays the manual for a command. `man ls` - Displays the manual for the "ls" command.
`man -k keyword` - Searches for commands related to a keyword.
